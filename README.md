# Dotfiles
Dotfiles for my build of Arch Linux

## Tools Used
- Terminal: Alacritty
- Window manager: I3
- Panel: Polybar
- Blur: Picom (exagerated here, i use a little less opacity)
- Text-editor: Visual Studio Code (i use neovim too)
